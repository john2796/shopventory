## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/john2796/shopventory.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. run local host

   ```js
   npm start

   ```

4. to run test
   ```js
   npm run test
   ```
